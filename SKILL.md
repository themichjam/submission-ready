## 1. JOURNAL OVERVIEW

**Full name:** International Journal of Epidemiology  
**Abbreviation:** IJE  
**Publisher:** Oxford University Press (OUP) on behalf of the International Epidemiological Association (IEA)  
**Submission system:** ScholarOne ManuscriptCentral — http://mc.manuscriptcentral.com/ije  
**Guidelines URL:** https://academic.oup.com/ije/pages/general_instructions

**Scope:** IJE is the official journal of the IEA, publishing epidemiological advances across communicable and non-communicable disease, health services research, preventive medicine, and new epidemiological and statistical methods. It is essential reading for researchers, teachers, and practitioners in epidemiology and public health worldwide. The journal is published six times yearly and is online-only; all articles are free to view 12 months after publication.

**Word count exclusions for all article types:** The word count always applies to main text only, excluding Abstract, Key Messages or Key Features, Declarations, References, Tables, Figures, and Supplementary material.

### Original Article — additional notes

Original Articles must include:
- A **structured abstract** of 250 words in four sections: **Background, Methods** (include study period and sample size), **Results** (include main results with estimates and confidence intervals), **Conclusions**.
- **Key Messages:** A summary in exactly 3 succinct bullet points, each a single complete sentence summarising the main messages of the paper.

### Cohort Profile — required sections (in order)

1. Title (must start "Cohort Profile:" followed by cohort name in full with acronym in parentheses)
2. Author List (affiliations; corresponding author marked with *)
3. Why was it set up?
4. Who is in the cohort?
5. What has been measured?
6. What has it found? Key findings and publications
7. What are the main strengths and weaknesses?
8. Can I get hold of the data? Where can I find out more?

**Eligibility requirements:** Baseline data collection must be complete with results beyond descriptive statistics; prospective follow-up completed or funded; >5000 participants at baseline preferred; at least one author must be a researcher involved with the data resource and available as contact.

**Pocket Profile** (for print, ~500 words): Cohort purpose; Cohort Basics; Follow-up and attrition; What has been measured? Key findings; Strengths and weaknesses; Data access.

### Data Resource Profile — required sections (in order)

1. Title (must start "Data Resource Profile:")
2. Author List
3. Data Resource Basics (area, dates, number of units/groups/individuals)
4. Data Collected (source data, sampling strategy, data type, ethical clearance, funding)
5. Data Resource Use (illustrative examples of use or publications)
6. Reasons to be cautious (limitations, generalisability)
7. Data Resource Access — includes "Profile in a nutshell" box at the end (≤200 words, written as brief complete sentences covering: why set up / unique features; dates, area, number covered; source data, sampling, data type; main data categories)

**Eligibility requirements:** Data access must be open or available via straightforward application; at least one wave of data collected and made available.

### Software Application Profile — additional notes

- IJE accepts: desktop applications, web browser-based services, server/command-line applications, plugins.
- Software must be non-trivial, require limited installation effort, and be compatible with multiple operating systems or hardware architectures where applicable.
- Web services must not require user registration.
- Title must include the software application name and concisely describe its use.
- Avoid overly technical jargon in abstract, introduction, example usage case, and discussion.

## 3. REQUIRED MANUSCRIPT STRUCTURE

### Original Article — section order

1. **Title page** — title (short, specific, no subtitles); running head (short, on title page directly below title); author names and affiliations; word count; corresponding author details (postal + email; one corresponding author only; marked with *); joint first/senior authors marked with superscript symbol and footnote if applicable
2. **Abstract** — 250 words; structured (Background / Methods / Results / Conclusions)
3. **Key Messages** — 3 bullet points, each a complete sentence
4. **Main text** — Introduction, Methods, Results, Discussion (or equivalent; no mandatory sub-headings beyond these)
5. **Declarations** (in the following order):
   - Ethics approval (in Methods, or as declaration if required)
   - Supplementary data (if applicable): state "Supplementary data are available at IJE online"
   - Funding — mandatory; sentence begins "This work was supported by…"; placed **before** Acknowledgements; if none: "Funding: None"
   - Acknowledgements — optional; consultants or minor contributors; must not include anyone who does not consent to being named
   - Author contributions — detailed statement per author (e.g. "AB designed the study. CD collected the data…"); required for ≤25 authors in the manuscript; if >25 authors, provide as supplementary file and write "Author contributions are available as a Supplementary file at IJE online"
   - Conflict of interest — mandatory; placed **after** Funding and Acknowledgements; if none: "Conflict of Interest: None declared"
   - Data availability — mandatory (applies to all papers submitted from July 2022 onwards); all code underlying the paper must be made available; data sharing strongly encouraged where ethically feasible
   - AI tools disclosure (if AI was used in data collection, analysis, or image generation): disclose in Methods AND in Declarations
6. **References** — Vancouver style (see Section 5)
7. **Tables** — in separate file(s)
8. **Figures** — in separate high-resolution file(s)

### Formatting requirements (all article types)

- Main document submitted as editable Word file (NOT PDF), designated 'Main document'
- If LaTeX: submit as PDF initially; Word or LaTeX source files required at acceptance
- Double spaced; margins of 2.5cm
- All pages numbered
- Title page included in main document
- Word count included on title page
- Headings and titles of tables and figures in **sentence case** (capitalise first word and proper nouns only)
- **Footnotes are NOT permitted in the main body of papers** — footnotes may appear only in tables and figures
- Tables and figures may be included in the main document at initial submission, or provided as separate files; if accepted, must be submitted as separate files

## 4. ABSTRACT REQUIREMENTS

### Original Article
- **Word limit:** 250 words
- **Format:** Structured, with four mandatory headings in this exact order:
  1. **Background**
  2. **Methods** — must include study period and sample size
  3. **Results** — must include main results with estimates and confidence intervals
  4. **Conclusions**
- Abbreviations used in the abstract must be spelled out in full at first use in the abstract (separately from spelling out in the text)
- No citations in abstract (standard Vancouver practice; not explicitly stated for IJE — verify with journal)

### Cohort Profiles, Data Resource Profiles, HDSS Profiles, Cohort Profile Updates
- **No abstract** — if prompted during online submission, select or type N/A

### Software Application Profile
- Abstract present; avoid overly technical jargon; word limit NOT SPECIFIED — verify with journal

### Key Messages (Original Article only)
- Exactly 3 bullet points
- Each must be a single complete sentence
- These are a separate element from the abstract and appear after it

## 5. REFERENCE STYLE

**Style name:** Vancouver (modified IJE style)  
**In-text citations:** Arabic numerals in square brackets, e.g. [1], [2,3]; inserted at point of citation in text, tables, and figures  
**Reference list order:** Numbered sequentially in the order they first appear in the text  
**Author rule:** List surnames and initials of ALL authors when ≤6; when ≥7, list first 3 authors then add "et al."  
**Journal abbreviations:** Index Medicus style (abbreviated journal names)  
**DOI policy:** Include DOI where available; required when citing preprints  

### Worked examples in IJE Vancouver style

**1. Journal article — with DOI (≤6 authors)**
> Davey Smith G, Ebrahim S. 'Mendelian randomization': can genetic epidemiology contribute to understanding environmental determinants of disease? Int J Epidemiol. 2003;32:1–22. doi:10.1093/ije/dyg070

**2. Journal article — without DOI (≥7 authors)**
> Lawlor DA, Harbord RM, Sterne JA, et al. Mendelian randomization: using genes as instruments for making causal inferences in epidemiology. Stat Med. 2008;27:1133–63.

**3. Book**
> Rothman KJ, Greenland S, Lash TL. Modern Epidemiology. 3rd ed. Philadelphia: Lippincott Williams & Wilkins; 2008.

**4. Book chapter**
> Clayton D, Hills M. Analysis of follow-up studies. In: Armitage P, Colton T (eds). Encyclopedia of Biostatistics. Chichester: Wiley; 1998:1414–26.

**5. Website / online source**
> World Health Organization. Global Health Observatory Data Repository. https://www.who.int/data/gho (20 January 2024, date last accessed).

**6. Report / grey literature**
> Office for National Statistics. National Life Tables, United Kingdom: 2019–2021. Newport: ONS; 2022.

**7. Preprint** (with DOI — required for IJE preprint citations)
> Surname A, Surname B. Preprint title. medRxiv 2023. doi:10.1101/2023.01.01.123456

### Common errors to watch for
- Listing more than 3 authors for papers with ≥7 authors without using "et al." (must be exactly 3 + "et al.")
- Listing all authors even when there are >6 (must truncate at 3 + "et al." for ≥7 authors)
- Using full journal names instead of Index Medicus abbreviations
- Missing page ranges (first and last page required)
- Numbering references in alphabetical rather than order-of-appearance order
- Using (1) or superscript 1 in-text instead of [1]
- Omitting DOI when it is available
- Using short funding agency names (e.g. "NCI" instead of "the National Cancer Institute at the National Institutes of Health")

## 6. FIGURES, TABLES & SUPPLEMENTARY MATERIAL

### Figures

**Accepted formats:** eps, tiff, png, jpg — do NOT submit figures in PDF, docx, or pptx format  
**Minimum resolution:** 300 dpi at a minimum width of 300 mm (equivalent to 3600 pixels wide)  
**Colour policy:** Photos reproduced free in colour online; black & white in print. [Colour charges for other figures — NOT SPECIFIED — verify with journal]  
**Numbering:** Consecutive Arabic numerals (Figure 1, Figure 2, etc.)  
**Legends:** Required; must be self-contained so the figure can be interpreted independently of the text; include meaningful title, legend, and adequate footnotes  
**Abbreviations in figures:** All abbreviations must be spelled out in full in a footnote to the figure (even if already defined in text)  
**Labels:** Provide axis labels of legible size; note that graphics programs often produce figures with labels that become illegible after resizing  
**Citation in text:** Each figure must be cited at an appropriate place in the text  
**At submission:** Figures may be included in the main document or provided as separate files; if accepted, must be provided as separate high-resolution files  

**Note on composite figures / panels:** NOT SPECIFIED in guidelines — verify with journal.

### Tables

**Format:** Editable Word or Excel format (NOT PDF)  
**Placement:** Separate file(s) from main text (may be in main document at initial submission; separate file required at acceptance)  
**Numbering:** Consecutive Arabic numerals  
**Titles:** Sentence case; meaningful; self-contained with adequate footnotes  
**Abbreviations:** All abbreviations spelled out in footnotes  
**Vertical rules:** Must NOT be used  
**Sub-headings in tables:** Use separate table rows for each line of data, including sub-headings; do NOT use paragraph returns to separate data items  
**Citation in text:** Each table must be cited at an appropriate place in the text  

### Supplementary / Online-only Material

**What is accepted:** One or more supplementary files for online-only publication  
**Designation:** Each file must be labelled "Supplementary file (for online publication only)" when uploaded  
**Reference in manuscript:** Add "Supplementary data are available at IJE online" under the heading "Supplementary data" in the Declarations section  
**Author contributions >25 authors:** Provided as a supplementary file  
**Size / page limits:** NOT SPECIFIED — verify with journal  
**Labelling convention:** NOT SPECIFIED in main guidelines — verify with journal; standard practice is Supplementary Table S1, Supplementary Figure S1, etc.  
**Abbreviations in supplementary material:** Must be spelled out at first use in supplementary material (separately from the main text)

## 7. MANDATORY DECLARATIONS

All of the following declarations must appear at the end of the manuscript in the Declarations section, in the order specified below. Two mandatory FORMS must also be submitted at the point of initial submission (see Section 7a).

### Funding (mandatory — placed BEFORE Acknowledgements)
- Required for all manuscripts; if no funding received, write: **"Funding: None"**
- Opening sentence must be: **"This work was supported by…"**
- Use the full official name of every funding agency, e.g. "the National Cancer Institute at the National Institutes of Health" or "National Institutes of Health" — NOT "NCI", "NCI at NIH", or other abbreviations
- Follow the Research Information Network (RIN)-approved list of agency names
- Funding section must be placed **before** any Acknowledgements section

### Acknowledgements (optional — placed AFTER Funding)
- Optional section
- May include consultants and minor contributors who did not meet authorship criteria
- Must not include persons who have not consented to being named
- Good practice to confirm consent of any named person

### Author contributions (mandatory)
- Required for up to 25 authors: detailed statement for each author naming their specific contributions (e.g. "AB designed the study. CD collected and cleaned the data. EF conducted the statistical analysis…")
- If >25 authors: add as a supplementary file designated "for online publication only" and write in the manuscript: "Author contributions are available as a Supplementary file at IJE online"
- No specific taxonomy (e.g. CRediT) is mandated, but contributions must be specific and detailed

### Conflict of interest (mandatory — placed AFTER Funding and Acknowledgements)
- In-manuscript statement required summarising all aspects of any conflicts declared on the COI Form
- If no conflict: **"Conflict of Interest: None declared"**
- The COI Form (separate mandatory document) must also be submitted at initial submission (see Section 7a)

### Data availability (mandatory — all papers submitted from July 2022)
- A Data Availability Statement must be included
- All software code underlying the paper must be made available to readers (mandatory)
- All data on which the conclusions rely should be made available where ethically feasible (strongly encouraged)
- Preferred approaches: data presented in the manuscript or supplementary material, or deposited in a public repository; code in supplementary material or public repository/code sharing platform
- See the OUP Research Data Policy page for recommended repositories

### Ethics approval (required where applicable)
- Must state the ethics approval body and approval reference number (or equivalent)
- Participant consent must be addressed where relevant
- Typically appears in the Methods section of the paper, not as a separate declaration — but must be present

### Supplementary data (required if applicable)
- Write: "Supplementary data are available at IJE online"
- Placed within the Declarations section

### AI tools disclosure (required if AI tools were used)
- If AI tools (e.g. ChatGPT, large language models) were used in data collection, analysis, image/graphical element generation, or manuscript preparation:
  - Disclose which AI tools were used and how, in the **Methods** section
  - Also declare in the **Declarations** section
- AI tools cannot be listed as authors
- Authors remain fully responsible for all content, including AI-generated portions

### 7a. Mandatory submission forms (separate from manuscript)
Two forms must be uploaded at initial submission. **Submission cannot be completed without them.**

**Authorship Form**
- Signed by the corresponding author on behalf of all authors
- Corresponding author is responsible for ensuring all listed authors meet ICMJE criteria and that no eligible contributor is omitted
- Available to download from the IJE guidelines page

**Conflict of Interest (COI) Form**
- Signed by the corresponding author on behalf of all authors
- Complete only ONE of Section I or Section II on page 2
- Must be submitted even if there is no conflict of interest

**Licence to Publish Form**
- Required only at acceptance (not at submission)
- Completed online via link sent by OUP

| Qualitative studies | NOT SPECIFIED | verify with journal |

Whether the completed checklist must be uploaded as a separate file is NOT EXPLICITLY SPECIFIED in the main guidelines — verify with the editorial office. ## 9. STATISTICAL REPORTING REQUIREMENTS

- **"Statistically significant" is banned.** Do not use "statistically significant", "significant", or statements such as "findings at P<0.05 were considered significant" in any section of the paper (including Methods, Results, and Discussion).
- **Effect estimates** must be reported with **confidence intervals**.
- **Exact P values** must be reported (not "P<0.05" or "P=NS").
- **P symbol:** Use **P** (upper case, italicised) for P values throughout.
- **n symbol:** Use **n** (lower case, italicised) for number of subjects.
- Statistical symbols are automatically typeset in italics; do not underline them except to prevent ambiguity when an isolated letter appears in text.
- Symbols must not be used to start a sentence.
- **Software disclosure:** NOT EXPLICITLY REQUIRED in guidelines — verify with journal, but best practice is to name the statistical software and version used.
- For statistical papers: carefully distinguish upper/lower case, ordinary vs bold-faced letters, certain Greek vs Roman letters, and subscripts/superscripts/ordinary symbols.

## 10. LANGUAGE, STYLE & FORMATTING

**English standard:** **UK English** (NOT US English). This is explicit in the guidelines.  

**Abbreviations:**
- Keep abbreviations to a minimum; avoid non-standard abbreviations
- All abbreviations must be spelled out in full with the abbreviation in parentheses at **first use** in each of: (a) the Abstract, (b) the Key Messages or Key Features, (c) the main text, (d) each table or figure (in a footnote), and (e) any supplementary material
- Thereafter the abbreviation may be used
- Abbreviations must NOT appear in: titles, table or figure titles, headings, or sub-headings
- Any abbreviation appearing in a table or figure must be defined in a footnote to that table or figure — even if already defined in the text

**Numbers:**
- Numbers from one to nine: spell out in the text
- Numbers above nine: Arabic numerals
- Numbers followed by a unit: always use numerals (e.g. 5 mg, 3 years)
- Numerals must not start a sentence
- Numbers up to 9999: no separator (e.g. 9999)
- Numbers from 10 000 onwards: use a space as separator (e.g. 10 000, 125 600)

**Units:**
- All measures reported in **SI units**, followed by traditional units in parentheses in the text
- **Exceptions:** blood pressure in **mmHg**; haemoglobin in **g/dL**
- Per cent written as **%** throughout (NOT "per cent")

**Age groupings:** Mid-decade groupings required (e.g. 35–44 or 35–39, 40–44); NOT full decades (e.g. do NOT use 30–39)

**Titles:**
- Short and specific; use a descriptive phrase with keywords; do NOT use a complete sentence
- Do NOT include excessive detail (e.g. sample size, methods) unless methods are the primary focus
- **No subtitles**

**Jargon:** Should be avoided throughout

**Footnotes in main body:** NOT PERMITTED — use references instead; footnotes may appear only in tables and figures

**Sentence case:** All headings and titles of tables and figures must use sentence case (capitalise first word and proper nouns only; do not title-case)

**Tobacco industry:** The IJE will not consider papers funded to any extent by the tobacco industry

**LMIC rule:** Papers using data, infrastructure, or personnel from any low- or middle-income country will not be published unless at least one author is a national of that country

**Authorship changes:** After submission, authorship changes (list, order, or corresponding author) are not permitted without substantive justification and agreement of editor and all co-authors

## 12. DRAFT AUDIT INSTRUCTIONS

When the user provides a manuscript draft (pasted text, uploaded file, or described content), follow these steps in order.

### Step 1 — Summarise the draft
In 3–5 sentences, describe:
- The study design and methods (e.g. retrospective cohort study using linked administrative data; systematic review and meta-analysis)
- The main research question or objective
- The approximate scope (number of participants, time period, data sources, outcomes)
- What the paper argues or concludes

### Step 2 — Identify the submission type
Based on the draft content, identify:
- Which IJE article type best fits this manuscript (Original Article, Cohort Profile, Data Resource Profile, SAP, etc.)
- The applicable word limit for that type (main text only, excluding Abstract, Key Messages, Declarations, References, Tables, Figures, Supplementary material)
- Which reporting guideline applies based on study design (STROBE / RECORD / PRISMA / MOOSE — see Section 8)
- State any uncertainty if the article type is ambiguous, and ask the user to confirm before proceeding

### Step 3 — Run the compliance audit
Work through the Pre-Submission Checklist systematically. For each item:
- Mark ✅ **PASS** if the requirement is clearly met
- Mark ❌ **FAIL** if the requirement is clearly not met — state exactly what is wrong and what is needed
- Mark ⚠️ **UNCLEAR** if you cannot determine compliance from the draft alone — state what information is missing

Pay particular attention to the following IJE-specific issues that are frequently missed:
1. Use of "statistically significant" or "significant" — **banned by IJE**
2. UK vs US English (e.g. "analysed" not "analyzed", "behaviour" not "behavior", "colour" not "color")
3. Exact P values required — not "P<0.05"
4. Effect estimates must be accompanied by confidence intervals
5. Abbreviations defined separately in Abstract, Key Messages, text, each table/figure, and supplementary material — not just once overall
6. Footnotes in main body — forbidden
7. Numbers: mid-decade age groupings; space as separator from 10 000
8. Full official funding agency names (no abbreviations)
9. Author contributions must be specific and detailed, not generic
10. Data availability statement required for all submissions from July 2022
11. Two mandatory forms (Authorship Form + COI Form) required at submission
12. Title has no subtitle; no complete sentence; no excessive methodological detail
13. Headings and table/figure titles must be in sentence case
